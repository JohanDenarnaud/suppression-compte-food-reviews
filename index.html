Procédure de Demande de Suppression de Compte

Pour demander la suppression de votre compte, veuillez suivre les étapes suivantes :

  • Ouvrez l'application Food Reviews.
  • Connectez-vous avec votre email et votre mot de passe.
  • Allez dans les paramètres puis dans "Modifier votre profil".
  • Cliquez sur "Supprimer mon compte".
  • Si vous rencontrez des problèmes, vous pouvez contacter notre support à l'adresse suivante : johandenarnaud@hotmail.fr.

Données Supprimées ou Conservées

Lorsque vous demandez la suppression de votre compte, les types de données suivants seront supprimés :

  • Données personnelles (nom et adresse e-mail).
  • Les différentes "Remarques" liés aux avis.

Cependant, certaines données peuvent être conservées conformément aux obligations légales ou pour d'autres raisons légitimes.
